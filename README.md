# Rock Paper Scissors – Python

A simple command-line Rock Paper Scissors game built using Python.  
This project was created to practice Python fundamentals such as conditionals, loops, functions, and user input.

## Features
- User vs Computer gameplay
- Random computer choices
- Input validation
- Score tracking
- Beginner-friendly code structure

## Tech Used
- Python 3

## What I learned
- Writing clean and readable Python code
- Using if-elif-else logic
- Working with the random module
- Basic refactoring and code organization

## How to Run
1. Clone the repository
2. Open the project in VS Code or terminal
3. Run:
   python main.py
